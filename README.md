# MazeRunner

## Installation

Pour installer MazeRunner, veuillez télécharger l'archive zip de la branche `dev` du répertoire. Extrayez-la à l'endroit que vous souhaité.

## Lancement

Pour lancer MazeRunner, ouvrez le fichier `code/interface/home.html` du répertoire du projet avec le navigateur de votre choix.

## Exécution des tests de performance sur les algorithmes

Pour exécuter les tests de performance sur les algorithmes, placez vous à la racine du répertoire. Ensuite, exécutez la commande `node --expose-gc code/Test_perf/Perf_algo.js` en vous assurant que vous ayez bien `node` d'installé. Une version 20.18.1 de `node` suffit à exécuter les tests. L'exécution entière des tests peut durer quelques instants.
